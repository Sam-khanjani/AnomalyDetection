# Industrial Anomaly Detection - Master Thesis (Politecnico di Torino)

This project is part of my **Master Thesis at Politecnico di Torino**. It is based on [EfficientAD](https://github.com/nelson1425/EfficientAD), an **unofficial implementation** of the paper:  
[EfficientAD: Accurate and Efficient Anomaly Detection for Industrial Applications](https://arxiv.org/abs/2303.14535).
[CDO: Collaborative Discrepancy Optimization for Reliable Image Anomaly Localization](https://arxiv.org/abs/2302.08769).

## 📌 About This Project

Anomaly detection is essential for identifying defects in industrial products. This project applies **EfficientAD** to industrial datasets RealIAD and MVtecCAD, evaluating its performance and potential improvements.

### Features:
- Implements **EfficientAD** for industrial anomaly detection.
- Works with real-world industrial datasets.
- Benchmarks performance against other detection methods.
- Avoid Overgeneralization with Collaborative Discrepancy Optimization
